A classic Sudoku game built with Python and Pygame, following MVC architecture.
Includes adjustable settings and a built-in Sudoku solver using recursive backtracking.

The file sudoku.csv contains the first 10,000 Sudoku puzzles from the following dataset:
9 Million Sudoku Puzzles and Solutions:
https://www.kaggle.com/datasets/rohanrao/sudoku?resource=download
